## Jekyll, Webpack, Vue

**INSTALL**

```
yarn install
gem install jekyll bundler
bundle install
```

**DEV**

```
yarn run dev
```

**BUILD**

```
yarn run build
```
(Site build is in ```/dist``` folder)

**SERVE**
```
yarn run serve
```

**BUILD & SERVE**
```
yarn run build:serve
```
